How to set up sync ▸
  1. Create a free account at github.com/signup (skip if you have one)
  2. Go to github.com → Settings → Token (click here — pre-filled)
  3. Scroll down, click Generate token
  4. Copy the token (starts with ghp_) — shown only once!
  5. Paste it in the token field above, pick a username, hit Save
  6. Use Push ↑ to upload, Pull ↓ to download on another device
